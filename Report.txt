Ανάπτυξη Project BookManagement
GitHub repository: https://github.com/ArviHoxha7/BookManagement

Για την εμφάνιση των dummy entries αναζητήστε με το keyword: Stories

1. Δημιουργία του backend
Αρχικά, δημιούργησα τον backend server χρησιμοποιώντας το Node.js και το Express. Ο server αυτός διαχειρίζεται μια βάση δεδομένων SQLite για την αποθήκευση των βιβλίων. 
Χρησιμοποίησα την εντολή `npm init -y` για να δημιουργήσω το αρχείο package.json και εγκατέστησα τα απαραίτητα πακέτα με την εντολή `npm install express sqlite3 body-parser`.

2. Δημιουργία της βάσης δεδομένων
Στον κώδικα του server (server.js), δημιούργησα ή σύνδεσα μια βάση δεδομένων SQLite με το όνομα books.sqlite. Δημιούργησα έναν πίνακα books για την αποθήκευση των βιβλίων 
με πεδία για τον συγγραφέα, τον τίτλο, το είδος και την τιμή του βιβλίου.

3. Υλοποίηση των endpoints
Υλοποίησα δύο endpoints στον server:
   - Ένα endpoint POST στη διαδρομή `/books` για την καταχώρηση νέων βιβλίων. Ο server λαμβάνει τα δεδομένα του βιβλίου σε μορφή JSON και τα αποθηκεύει στη βάση δεδομένων.
   - Ένα endpoint GET στη διαδρομή `/books/:keyword` για την αναζήτηση βιβλίων με βάση μια λέξη-κλειδί στον τίτλο. Ο server επιστρέφει τα βιβλία που ταιριάζουν σε μορφή JSON.

4. Δημιουργία του frontend
Δημιούργησα έναν φάκελο public για τα αρχεία του frontend και μέσα σε αυτόν τα αρχεία index.html, styles.css και script.js. Στο index.html δημιούργησα δύο φόρμες: μία για την 
καταχώρηση νέων βιβλίων και μία για την αναζήτηση βιβλίων.

5. Στυλ
Για να κάνω την εμφάνιση της σελίδας πιο ευχάριστη, πρόσθεσα στυλ στο styles.css. Χρησιμοποίησα flexbox για να κεντράρω το περιεχόμενο, πρόσθεσα σκιές και στρογγυλεμένες 
γωνίες στα στοιχεία της σελίδας και εφάρμοσα εφέ hover στα αποτελέσματα αναζήτησης.

6. Λειτουργικότητα με JavaScript
Στο script.js υλοποίησα τη λειτουργικότητα των φορμών χρησιμοποιώντας καθαρή JavaScript. Για τη φόρμα καταχώρησης, έστειλα τα δεδομένα στον server χρησιμοποιώντας τη μέθοδο 
fetch με αίτημα POST. Για τη φόρμα αναζήτησης, έκανα fetch με αίτημα GET και εμφάνισα τα αποτελέσματα στη σελίδα.

Αυτά είναι τα βασικά βήματα που ακολούθησα για την ανάπτυξη του project. Η εφαρμογή επιτρέπει στον χρήστη να καταχωρεί νέα βιβλία και να τα αναζητά με βάση τον τίτλο τους, 
ενώ η βάση δεδομένων αποθηκεύεται σε αρχείο SQLite για μόνιμη αποθήκευση των δεδομένων.
